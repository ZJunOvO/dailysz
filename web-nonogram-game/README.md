# 🎮 卷宝专属数织游戏 Web版

> 为卷宝定制的浪漫数织解谜游戏，充满爱意的每一个细节 💕

## 🌟 项目特色

- **个性化定制**：专为卷宝设计，默认用户名"卷卷大王"
- **现金奖励系统**：完成关卡获得 ¥0.52-5.20 现金奖励
- **智能奖励计算**：
  - 难度系数加成 (0.5x-2.0x)
  - 连胜奖励递增 (最高3倍)
  - 特殊日期奖励 (生日5倍，纪念日3倍)
- **关卡系统**：前51关自动解锁，可在设置中切换
- **浪漫元素**：特殊日期检测和专属祝福

## 🎯 核心功能

### 游戏玩法
- **数织解谜**：经典Picross/Nonogram玩法
- **操作方式**：
  - 左键点击：填充/清空格子
  - 右键点击：标记X/清除标记
- **辅助功能**：
  - 💡 智能提示
  - ↶ 撤销操作  
  - ↷ 重做操作
- **难度分级**：入门→简单→中级→困难→专家

### 奖励系统
- **基础奖励**：¥0.52-5.20 根据难度浮动
- **连胜加成**：
  - 1-2连胜：1.0x
  - 3-5连胜：1.2x
  - 6-10连胜：1.5x
  - 11-20连胜：2.0x
  - 21+连胜：3.0x
- **特殊日期**：
  - 04-22 (卷宝生日)：5倍奖励 🎂
  - 12-28 (菌子生日)：5倍奖励 🎂
  - 09-16 (恋爱纪念日)：3倍奖励 💕
